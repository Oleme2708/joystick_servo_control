# joystick servo control

A PlatformIO project for joystick servo controller

# Hardware
- arduino uno
- 4 servo motors (MG995)
- 2 analog joysticks
- 5V supplies (This project used breadboard power supplies module)

## Software
- VS Code
- PlatformIO

## How to Use
Open this project in VS Code with PlatformIO installed, then build and upload the code to the board.
